# 说明
---

 文件名        |     说明
:------------: | :-----------------:
build-function | 內建函数，系统记录
data-struct    | 内置数据结构
class-object   | 类和对象
os-sys         | 系统相关
other          | 其他

---

  `build-function` 负责记录零散的內建函数，给出用法和例子。按照函数名首字母排序。
  `data-struct` 负责记录 python 内置的数据对象，和相关方法。
  `class-object` 负责记录类相关的知识和用法
  `os-sys` 负责记录系统文件的操作
  `other` 负责记录一些零散的心得体会
